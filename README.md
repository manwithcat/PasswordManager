Password Manager - это приложение для безопасного управления и хранения паролей, разработанное на Java 17 с использованием современного графического интерфейса. Приложение позволяет пользователям регистрироваться, аутентифицироваться и управлять паролями для различных онлайн-сервисов с поддержкой нескольких криптографических методов шифрования.

Основные функции

- Система регистрации и аутентификации пользователей с криптографическим хешированием пароля (MD5)
- Добавление, редактирование, просмотр и удаление сохраненных паролей
- Поддержка пяти методов шифрования для хранения паролей:
  - BASE64 (кодирование, минимальная защита)
  - MD5 (необратимое хеширование)
  - AES_WITH_SALT (XOR шифрование с солью)
  - FEISTEL (16-раундовая сеть Фейстеля)
  - PLAINTEXT (без защиты, только для тестирования)
- Выбор метода шифрования для каждого пароля отдельно
- Безопасное хранение данных в SQLite базе данных
- Интуитивный графический интерфейс на основе Swing с современным темным дизайном (FlatLaf)
- Защита от SQL-инъекций через использование PreparedStatement
- Валидация всех пользовательских вводов
